# PolyhedralTodoList

<img src="images/polyDerived.jpg" alt="Polyhedral Icon" align="right" />
You can manage Google Tasks as merged TODO tasks on Android.

### Details
Merged TODO Tasks
* Add a new task
* Move up/down a task

TODO Tasks
* Add a new task
* Move up/down a task
* Edit titles

TODO List
* Add a new list
* Edit titles
